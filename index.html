Bitcoin Public Key to Receiving Address Calculator


Need to convert a bitcoin compressed public key into a receiving address? Probably not. But this page will be here if you ever do.

This page was mainly built for education purposes. However, one example of when you might need this is if you're ever required to demonstrate ownership of a receiving address without sharing your seed phrase, private key (extended or individual), extended public key, or by signing a message.

You cannot enter a private key into this site on purpose (as it's a security risk). If you're concerned about entering your public key on this page, then you can download the site from Github and run it offline.

Contact me on Twitter (@mission_bitcoin) or submit an issue/pull request on Github.
